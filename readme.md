android-spaceinvaders 
==============

Ein Android Spiel für die "Connection Machine".

Viele böse Aliens wollen unsere schöne kleine Welt besetzen.
Fliegen Sie mit ihrem kleinen Raumschiff den bösen Invasoren entgegen und vernichten Sie sie!
Retten Sie die Welt!
 
Merkmale
==============
- Steuerung durch Neigung mit dem Smartphone
- Feuern durch das tippen auf den Bildschirm des Smartphones
- Anzeige der Punkte und Leben auf dem Bildschirm des Smartphones
- Abspeichern der Höchst-Punktzahl
- Spielbar auf einer "Connection Machine" bzw auf einer 24x24 LED-Matrix
- Verbindung mit der "Connection Machine" über Bluetooth
- ca 10 Gegner in einer "Stage"
- Gegner bewegen sich und werden schneller
- Gegner schießen auf den Spieler
